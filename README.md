# Guided Project - Calculate and Print Student Grades

First requirement for the C# Software Development pathway from Code:You. This is a lesson from [learn.microsoft.com](https://learn.microsoft.com/en-us/training/modules/guided-project-calculate-print-student-grades/) that focuses on the basics:

- Experience using the .NET editor
- Experience with basic C# syntax rules
- Performing math operations on variables
- Creating literal values
- Declaring variables of basic data types like `string`, `int`, and `decimal`
- Experience with string concatenation and string interpolation
